How to enable Accessibility permissions for Memorian on MacOSX.

If you use Mac OS X, Memorian requires accessibility permissions to be enabled so it can detect when you copy a content. Without these additional permissions Memorian cannot capture copy content details.

Does this mean that Memorian has control of my computer?

No. When you enter ⌘+C or ⌘+X, you're simply given Memorian the permission to save your clipboard information into another local memory on your computer. This will help you save your previouse copied contents in case you need them in the future.

What happens if I don't agree to the accessibility permissions?

The app functionality relys on this accessibility; hence, it is neccessary to enble accessibility to use this app.

How do I enable Accessibility permissions on my Mac?

Step 1:

Open System preference.

Step 2:

Go to Security & Privacy.

Step 3:

Click Privacy tab, and select Accessibility from the left list.

Step 4:

Click the lock icon, and unlock by entering the password.

Step 5:

If you don't see the app in the list, click the plus mark to add the application to the list.

Step 6:

Select Memorian, then click Open to add the application to the list.

Step 7:

Make sure Memorian is selected. If the checkbox is not selected, click the checkbox.

Step 8:

Close System Preferences, and close Memorian. Re-open Memorian.
